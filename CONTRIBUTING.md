# Contributing

This repo documents a design practice in progress. Entries and skills are published weekly.

**Found an issue with a skill?** Open an issue with the skill name and what broke.

**Have a behavior pattern to suggest?** Open an issue with: the behavior name, the context it applies to, and why it matters. I'll consider it for a future entry.

**Want to adapt a skill for your domain?** Fork it. The MIT license means it's yours to use, modify, and build on. If you build something useful, I'd like to see it.
